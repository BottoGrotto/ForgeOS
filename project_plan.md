# ForgeOS Project Plan

This tracked plan is intentionally stable and high level.

ForgeOS is an operating system for autonomous AI organizations: operators create isolated Forges, coordinate Executive AI and workers, run provider-backed operations, and inspect runtime state, artifacts, workspace files, logs, and usage.

The durable setup, environment, runtime, provider, and verification guidance lives in `README.md`.

Transient implementation planning, session handoffs, and short-lived task breakdowns should stay outside tracked source unless they are promoted into durable documentation.
